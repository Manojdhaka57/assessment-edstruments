### Installation

1. Clone the repository

```bash
git clone https://github.com/Manojdhaka57/assessment-edstruments.git
```

2. Navigate to project directory

```
cd assessment-edstruments
```

3. Install dependencies

```
npm install
```

4. Start the development

```
npm start
```
